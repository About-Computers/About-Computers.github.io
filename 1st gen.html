1st Generation

Machine Language

In Computer's First Generation, Programmers Had To Use Machine Language Because No Other Option Was Available.

  1. Machine Language Is Low Level Language.
  2. It Consists Of Entirely 1s And 0s.
  3. It Is The Only Programing Language That The Computer Understands Directly Without Any Translation.
  4. The Programs Written In Machine Language Have The Advantage Of Very Fast Execution Speeds And Efficient Use Of Primary Memory.
  5. Each Proccesor Requires Its Own Machine Language.
    For This Reason, Machine Language Is Said To Be Machine-Dependent Or Hardware-Depentdent
  6. Use Of Machine Language Is Very Tedious, Difficult And Time Consuming.

3rd Generation

High-Level Language

Procsdural Languages Such As FORTRAN (Formula Translation), ALGOL (Algorithmic Language), COBOL (Common Business Oriented Language), BASIC (Beginner's All Purpose Symbolic Instruction Code), PASCAL, C, C++ And Java Are Considered As Third Generation Languages.

  1. In Procedural Languages, A Program Has A Sequence Of Instructions That Tells A Computer What To Do Step By Step.
  2. In These Languages, Common English Words Were Used Rather Than Abbreviated Mnemonics.
  3. Because Computers Understand Only Machine Language, High-Level Languages Must Be Translated Into Machine Code Prior To Execution.
  4. Translation Is Accomplished By Language Translators Such As Compiler And Interpreter.
  5. These Are Closer To Human Languages.
  6. They Are Easier To Read And Write.

4th Generation

High-Level Language

Non-Procedural Languages Such As SQL, Oracle Reports, PostScript, Informix-4GL And Borland Delphi Are Considered As Fourth Generation Languages.
These Types Of High-Level Languages Allow The Users To Define 'What Is Required' Rather Than 'How It Is To Be Done'.

  1. These Are Much Closer To Human Languages Than Third Generation Languages.
  2. These Are Designed To Reduce The Overall Time, Effort And Cost Of Software Development.
  3. These Languages Are Common In Database Applications As Query Languages, Report Generators, And Data Manipulation Languages.
  4. They Allow Users And Programmers To Interrogate And Access Computer Databases Using Statements That Resemble Natural Language.
